# Course: @vasic-digital/catalogizer-api-client — Introduction

## Module Overview

This module is the official TypeScript client for the Catalogizer API. It abstracts all HTTP communication, token management, and error handling so your application code stays clean and type-safe.

## What You Will Learn

1. Creating a `CatalogizerClient` and configuring it
2. Authentication flow: login, token refresh, logout
3. Browsing and searching media entities
4. Managing collections and smart rules
5. Handling errors and the auth:expired event
6. Testing components that depend on the client

## Prerequisites

- TypeScript 5.0+
- Basic understanding of async/await and REST APIs
- `@vasic-digital/media-types` installed
